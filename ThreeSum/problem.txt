Given an array nums of n integers, there are elements a, b, c in nums such that a + b + c = 0? Find all unique triplets in the array which gives the sum of zero.

Note:

The solution set must not contain duplicate triplets.

========
Example:
========

Input:
6
-1 0 1 2 -1 -4

Output:
-1 0 1
-1 -1 2

Note: List of triplet can be printed in any order
